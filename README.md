# Naruto Roulette

A dynamic Naruto-themed roulette game built with Angular. Navigate through different life phases as a shinobi, make decisions that affect your progression, and unlock achievements and perks as you grow stronger.

## Features

- **Phase-Based Progression**: Progress through Academy → Genin → Chunin → Jonin → Kage phases
- **Dynamic Wheel System**: Wheel contents change as you complete events, gradually forcing you toward the next phase
- **20+ Canon Characters**: Encounter Naruto, Sasuke, Kakashi, and many more with phase-scaled difficulty
- **Achievement System**: Unlock achievements that grant perks for future runs
- **Perk System**: Equip perks to modify gameplay difficulty and mechanics
- **Random Teammate System**: Get randomly assigned teammates and sensei each run
- **Death Mechanics**: Permadeath on certain encounters (no plot armor!)
- **8-Stat System**: Ninjutsu, Taijutsu, Genjutsu, Intelligence, Strength, Speed, Stamina, Seals
